Dado que es una web que consta de una sola página y no es nada compleja, intenté ahorrarme la mayor cantidad de "id"s y clases
a la hora de escribir el html para que sea vea más limpio y sea más fácilmente legible. A pesar de ello, sigue siendo sencillo
seguir el hilo del diseño teniendo el css al lado. De no ser así, o de contar con más elementos en la página, hubiera tirado
de usar más ids.

Ya que no es un proyecto con el que vaya a seguir o al menos no con la seriedad que podría tener una tienda real, consideré que
no sería relevante que se repitan los textos descriptivos de los productos enseñados.
